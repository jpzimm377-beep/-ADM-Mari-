# -ADM-Mari-
Bot de Discord focado em eventos automáticos e economia virtual. Possui sistema de Caça ao Tesouro, com pistas enviadas a cada 3 horas, sem envio de madrugada, misturando pistas verdadeiras e falsas. Premia apenas 3 vencedores com PixCoins, VIP Ouro e cargo exclusivo, aumentando o engajamento da comunidade.
